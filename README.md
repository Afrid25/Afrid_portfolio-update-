# MD AFRID FOISAL OPU - 3D Animated Portfolio

A sleek, 3D animated portfolio website showcasing my skills and projects as a CSE student at IIUC.

## Features

- **3D Animations**: Interactive Three.js powered 3D scene with rotating cube, torus, and particle effects
- **Responsive Design**: Optimized for all devices
- **Cyber Theme**: Dark cyberpunk aesthetic with neon green and red accents
- **Smooth Animations**: Fade-in effects and smooth scrolling
- **No Nested Folders**: Flat file structure for easy GitHub Pages hosting

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6+)
- Three.js for 3D graphics

4. Choose "main" branch and "/ (root)" folder
5. Save and wait for deployment
